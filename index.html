A short story on the birth of Siricoin

This is how our coin started

SiriCoin first started as a mineable token on polygon network, at the time we only had a webminer and there was no support for avr mining. Suddenly there was a huge bug that caused the network to crash and the users werent able to mine. So we tried a lot to fix the bug but, but it was impossible. Since then we began to create our own network for SiriCoin. After this we upgraded the old website to this new and clean lookling website. From there our journey of SiriCoin began, and still continuing...
